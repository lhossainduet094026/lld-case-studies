# LLD Case Studies

This repository contains hands-on Low-Level Design exercises to practice OOP principles, SOLID, design principle, design patterns, and system modeling in Java.

# 🎯 Objective

- Strengthen OOP fundamentals through implementation
- Practice system modeling using real-world scenarios
- Understand relationships like Association, Aggregation, and Composition
- Apply Dependency Injection and SOLID principles
- Improve clean code and design thinking

  # 🧠 Concepts Practiced

## OOP Fundamentals
- Association
- Aggregation
- Composition
- Inheritance
- Polymorphism

## Design Principles
- Dependency Injection (DI)
- SOLID Principles (coming soon)

## Design Patterns (planned)
- Strategy Pattern
- Factory Pattern
- Builder Pattern
- Observer Pattern

# 🚀 How to Run

1. Clone the repository
2. Open in IntelliJ IDEA / Eclipse
3. Navigate to each module
4. Run the main class for each example

---

# 📌 Key Learnings

- Object relationships in real systems
- How to design loosely coupled systems
- Difference between ownership vs association
- Importance of constructor-based dependency injection
- Writing clean, modular, and testable code
